# Meal Planner Application

## Project Overview

This project is a *meal planning application* designed to allow users to streamline their weekly meal planning. Users will be able to manage a collection of meals by adding, modifying, or removing meals as needed. Each meal will include various attributes, such as effort and number of ingredients, that allow the user to search for meals meeting specific criteria and generating a meal plan within user specified constraints. The purpose of this application is to simplify selecting meals and generating ingredient lists.

The intended users are individuals who want to spend less time planning meals for the week or eliminate the need to manually creating shopping lists for ingredients. This project is of interest to me since I spend a decent amount of time each month picking meals and preparing a grocery list. A custom application like this would save me time and effort.

**Planned Features**:
- Add, modify, and remove meals from a collection of meals
- Find and view meals based on search criteria
- Generate a meal plan and grocery list based on given constraints


## Instructions for End User

- You can view the meals already in your collection byt clikcing the View Meals button
- You can add meals to your meal collection by clicking the Add Meal button and inputting the fields of the new meal
- You can filter the meals you want to see by clicking the Filter Meals button, then entering the desired parameter, or leave any parameter as is to not filter by that parameter.
- You can remove meals from the collection by clicking the (-) icon for the meal on the View Meal list.
- You can edit meals from the collection by clicking the pencil icon for the meal on the View Meal list.
- You can locate my visual components on start up (logo) and the icon for the menu buttons.
- You can save the state of my application by clicking the Save/Load button, then selecting Save meal collection on the next panel.
- You can load the state of my application by clicking the Save/Load button, then selecting Load meal collection on the next panel.
